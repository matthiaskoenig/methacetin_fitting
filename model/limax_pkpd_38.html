PKPD model for methacetin based liver function tests.

Description

This model is a physiological pharmacokinetic/pharmacodynamics model (PKPD) for the absorption, distribution, metabolism and elimination of methacetin, bicarbonate and paracetamol encoded in SBML format.

The content of this model has been carefully created in a manual research effort.
